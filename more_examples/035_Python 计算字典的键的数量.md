# Python 计算字典的键的数量

## 题目描述
Python3 实例
在 Python 中，字典是一种非常常用的数据结构，它由键值对组成。有时我们需要知道字典中有多少个键，这可以通过使用len()函数来实现。

## 实例
### 代码
```python
my_dict={'name':'Alice','age':25,'city':'New York'}
 num_keys=len(my_dict)
 print(num_keys)
```
### 输出结果
```
3
```
