# Python 输出1到n的所有奇数

## 题目描述
Python3 实例
编写一个 Python 程序，输出从 1 到 n 的所有奇数。假设 n 是一个正整数。

## 实例
### 代码
```python
n=int(input("请输入一个正整数 n: "))
 foriinrange(1,n +1):
     ifi %2!=0:
         print(i)
```
### 输出结果
```
1
3
5
7
9
```
