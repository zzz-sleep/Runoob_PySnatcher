# Python 判断一个列表中是否包含特定元素

## 题目描述
Python3 实例
在 Python 中，我们可以使用in关键字来判断一个列表中是否包含特定元素。in关键字会返回一个布尔值，如果元素存在于列表中，则返回True，否则返回False。

## 实例
### 代码
```python
my_list=[1,2,3,4,5]
 element=3
 ifelementinmy_list:
     print(f"{element} 存在于列表中")
 else:
     print(f"{element} 不存在于列表中")
```
### 输出结果
```
3 存在于列表中
```
