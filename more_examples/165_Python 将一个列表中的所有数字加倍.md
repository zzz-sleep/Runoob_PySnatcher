# Python 将一个列表中的所有数字加倍

## 题目描述
Python3 实例
在 Python 中，我们可以通过列表推导式（List Comprehension）来快速地将一个列表中的所有数字加倍。列表推导式是一种简洁且高效的方式来创建新的列表。

## 实例
### 代码
```python
# 原始列表
 numbers=[1,2,3,4,5]
 # 使用列表推导式将列表中的每个数字加倍
 doubled_numbers=[num *2fornuminnumbers]
 # 输出结果
 print(doubled_numbers)
```
### 输出结果
```
[2, 4, 6, 8, 10]
```
