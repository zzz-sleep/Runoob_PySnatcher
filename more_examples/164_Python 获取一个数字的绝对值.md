# Python 获取一个数字的绝对值

## 题目描述
Python3 实例
在 Python 中，获取一个数字的绝对值非常简单，可以使用内置的abs()函数。这个函数接受一个数字作为参数，并返回它的绝对值。

## 实例
### 代码
```python
number=-10
 absolute_value=abs(number)
 print(absolute_value)
```
### 输出结果
```
10
```
