# Python 将一串数字转换为二进制表示

## 题目描述
Python3 实例
在 Python 中，我们可以使用内置的bin()函数将整数转换为二进制字符串。这个函数会返回一个以 "0b" 开头的字符串，表示这是一个二进制数。

## 实例
### 代码
```python
# 定义一个整数
 number=42
 # 使用 bin() 函数将其转换为二进制字符串
 binary_representation=bin(number)
 # 打印结果
 print(binary_representation)
```
### 输出结果
```
0b101010
```
