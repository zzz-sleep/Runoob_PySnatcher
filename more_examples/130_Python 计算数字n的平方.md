# Python 计算数字n的平方

## 题目描述
Python3 实例
这个简单的 Python 程序将计算一个数字 n 的平方。平方是指一个数乘以它自己。

## 实例
### 代码
```python
defcalculate_square(n):
     returnn * n
 # 示例使用
 number=4
 result=calculate_square(number)
 print(f"The square of {number} is {result}")
```
### 输出结果
```
The square of 4 is 16
```
