# Python 字符串查找特定字符的位置

## 题目描述
Python3 实例
在 Python 中，我们可以使用find()方法来查找字符串中特定字符的位置。find()方法会返回字符在字符串中第一次出现的索引位置，如果字符不存在，则返回 -1。

## 实例
### 代码
```python
# 定义一个字符串
 text="Hello, welcome to the world of Python."
 # 查找字符 'w' 的位置
 position=text.find('w')
 # 输出结果
 print("The position of 'w' is:",position)
```
### 输出结果
```
The position of 'w' is: 7
```
