# Python 将一个字典的键转换为列表

## 题目描述
Python3 实例
我们可以通过使用list()函数将字典的键转换为列表。字典的键可以通过keys()方法获取。

## 实例
### 代码
```python
my_dict={'name':'Alice','age':25,'city':'New York'}
 keys_list=list(my_dict.keys())
 print(keys_list)
```
### 输出结果
```

['name', 'age', 'city']
```
