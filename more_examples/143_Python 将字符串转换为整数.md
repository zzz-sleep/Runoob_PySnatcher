# Python 将字符串转换为整数

## 题目描述
Python3 实例
在 Python 中，我们可以使用内置的int()函数将字符串转换为整数。这个函数会尝试将字符串解析为一个整数，并返回相应的整数值。如果字符串不能被解析为整数，则会抛出ValueError异常。

## 实例
### 代码
```python
# 示例字符串
 str_num="123"
 # 将字符串转换为整数
 int_num=int(str_num)
 # 打印转换后的整数
 print(int_num)
```
### 输出结果
```
123
```
