# Python 使用列表推导式生成一个列表

## 题目描述
Python3 实例
列表推导式是 Python 中一种简洁且强大的工具，用于创建列表。它允许你在一行代码中生成列表，通常比使用传统的 for 循环更简洁和高效。

## 实例
### 代码
```python
squares=[x**2forxinrange(10)]
```
### 输出结果
```

[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```
