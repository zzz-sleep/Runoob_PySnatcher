# Python 将列表中的每个元素平方

## 题目描述
Python3 实例
我们可以使用 Python 的列表推导式来轻松地将列表中的每个元素平方。列表推导式是一种简洁的方式来创建列表，它允许我们在一个表达式中对列表中的每个元素进行操作。

## 实例
### 代码
```python
# 原始列表
 original_list=[1,2,3,4,5]
 # 使用列表推导式将每个元素平方
 squared_list=[x**2forxinoriginal_list]
 # 输出结果
 print(squared_list)
```
### 输出结果
```
[1, 4, 9, 16, 25]
```
