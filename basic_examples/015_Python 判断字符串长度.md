# Python 判断字符串长度

## 题目描述
Python3 实例
给定一个字符串，然后判断该字符串的长度。

## 实例 1：使用内置方法 len()
### 代码
```python
str="runoob"print(len(str))
```
### 输出结果
```

6

```
