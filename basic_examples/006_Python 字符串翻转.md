# Python 字符串翻转

## 题目描述
Python3 实例
给定一个字符串，然后将其翻转，逆序输出。

## 实例 1：使用字符串切片
### 代码
```python
str='Runoob'print(str[::-1])
```
### 输出结果
```

boonuR

```
## 实例 2：使用 reversed()
### 代码
```python
str='Runoob'print(''.join(reversed(str)))
```
### 输出结果
```

boonuR

```
