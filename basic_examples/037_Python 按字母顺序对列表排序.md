# Python 按字母顺序对列表排序

## 题目描述
Python3 实例
Python 按字母顺序对列表排序，可以使用以下两个方法：
sort() 方法实例：

## 实例
### 代码
```python
my_list=["apple","banana","cherry","date"]
 my_list.sort()# 按字母顺序排序
 print(my_list)
```
### 输出结果
```
['apple', 'banana', 'cherry', 'date']

```
## 实例
### 代码
```python
my_list=["apple","banana","cherry","date"]
 sorted_list=sorted(my_list)# 创建一个新的已排序列表
 print(sorted_list)
```
### 输出结果
```

['apple', 'banana', 'cherry', 'date']
```
