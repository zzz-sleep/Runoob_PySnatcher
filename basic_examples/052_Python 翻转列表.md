# Python 翻转列表

## 题目描述
Python3 实例
定义一个列表，并将它翻转。
例如：
```

翻转前 : list = [10, 11, 12, 13, 14, 15]
翻转后 : [15, 14, 13, 12, 11, 10]

```

## 实例 1
### 代码
```python
defReverse(lst):
     return[eleforeleinreversed(lst)]
 lst=[10,11,12,13,14,15]
 print(Reverse(lst))
```
### 输出结果
```

[15, 14, 13, 12, 11, 10]

```
