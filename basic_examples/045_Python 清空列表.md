# Python 清空列表

## 题目描述
Python3 实例
定义一个列表，并清空列表，可以使用 clear() 方法实现。

## 实例 1
### 代码
```python
RUNOOB=[6,0,4,1]
 print('清空前:',RUNOOB)
 RUNOOB.clear()
 print('清空后:',RUNOOB)
```
### 输出结果
```

清空前: [6, 0, 4, 1]
清空后: []

```
