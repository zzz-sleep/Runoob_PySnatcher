# Python 判断字符串是否存在子字符串

## 题目描述
Python3 实例
给定一个字符串，然后判断指定的子字符串是否存在于该字符串中。

## 实例
### 代码
```python
defcheck(string,sub_str):if(string.find(sub_str)== -1):print("不存在！")else:print("存在！")string="www.runoob.com"sub_str="runoob"check(string,sub_str)
```
### 输出结果
```

存在！
```
